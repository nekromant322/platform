Java pre-project. Практическая задача 2.2.2


Используя SpringBoot (!), магию, видео и текстовые курсы из интернета и видеокурс Алишева Курс по Спрингу (он не по спринг буту, но в целом много спринг фич)

Задание:
1. Создайте один контроллер, замаппленный на /cars.

2. Создайте модель Car с тремя произвольными полями.

3. Создайте список из 10 машин.

4. Создайте сервис с методом, который будет возвращать указанное число машин из созданного списка.

5. Создайте страницу cars.html. Реализуйте создание таблицы с машинами из сервиса с помощью thymeleaf.

6. При запросе /cars выводить весь список. При запросе /cars?count=2 должен отобразиться список из 2 машин, при /cars?count=3 - из 3, и тд. При count ≥ maxCar выводить весь список машин.

7. maxCar задается в .property
