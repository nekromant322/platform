В следующем степе потребуется скачать и установить Java Development Kit — набор инструментов для разработки программ на Java.
Вам пригодится следующая страница: www.oracle.com/technetwork/java. Можете сразу открыть ее в новой вкладке. При установке JDK отметьте пункт Source Code, чтобы были установлены исходники классов стандартной библиотеки. Удобно иметь их под рукой, чтобы заглядывать при необходимости.
